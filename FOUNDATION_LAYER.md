# Foundation Layer (Core Chain & Tokens)

## Etrna L1 / ETR Chain
- Custom EVM-compatible Layer 1 that underpins all Etrna activity.
- Serves as the settlement layer for contracts covering tokens, NFTs, billing, and governance.
- Hosts bridges, staking, governance, rewards, identity, and Universal Entitlements Framework (UEF) core contracts.

## $ETR – Governance / Gas / Value
- Native gas token of Etrna L1.
- Used for gas, protocol fees, subscription billing, marketplace fees, and bridge/swap fees.
- Staking drives user tiers, governance weight, and ecosystem multipliers.

## $VIBE – Rewards / Engagement Token
- Earned across the ecosystem (VibeCheck, UEF Vault, EtrnaVerse, EtrnaMusic, quests, and more).
- Spent on boosts, perks, upgrades, in-app utilities, and select marketplace actions.
- Designed as a high-velocity rewards currency tightly linked to daily activity.

## Etrna Universal Bridge & Swap Layer
- **EtrnaUniversalBridge** smart contracts plus backend relayers enable cross-chain transfers.
- **EtrnaSwapRouter** aggregates swaps across DEXs.
- Bridge and swap activity feed into the Entitlements & Rewards Engine as bonus multipliers.
